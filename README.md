# Portfolio Web Interactivo

Este repositorio contiene el código fuente de mi portfolio web personal, construido con **Next.js**, **React** y **Framer Motion**.

El objetivo de este proyecto es presentar mi perfil profesional y proyectos de una manera visualmente atractiva y dinámica, haciendo uso de animaciones ligadas al scroll.

## 🚀 Tecnologías Principales

- **Framework**: [Next.js](https://nextjs.org/)
- **Librería UI**: [React](https://reactjs.org/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Tipado**: [TypeScript](https://www.typescriptlang.org/)

## ⚙️ Cómo ejecutar el proyecto localmente

1. Clona el repositorio:
   ```bash
   git clone https://github.com/RubizZ/portfolio.git
   cd portfolio
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
